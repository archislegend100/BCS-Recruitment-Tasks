# BCS Secretary Recruitment Tasks

Submissions for the Brain and Cognitive Society (BCS) Secretary Recruitment at IIT Kanpur. The recruitment involved two independent open-ended tasks, each requiring implementation of a complete project.

## Tasks

### Task 1 — Dance of the Planets

An orbital mechanics simulation modelling the motion of planets under gravitational interaction. The notebook implements numerical integration of the equations of motion and visualises planetary trajectories, demonstrating concepts from classical mechanics and computational physics.

### Task 2 — VeritasVigil

A misinformation detection and fact-checking system. VeritasVigil takes a claim or news article as input and evaluates its credibility by cross-referencing it against reliable sources and applying natural language inference techniques. The accompanying PDF report details the design, methodology, and evaluation of the system.

## Repository Structure

| File | Description |
|---|---|
| `Dance of the Planets.ipynb` | Orbital mechanics simulation and visualisation |
| `VeritasVigil.ipynb` | Misinformation detection pipeline |
| `VeritasVigil Final Report.pdf` | Technical report for the VeritasVigil system |

## Requirements

Dependencies vary by task. The general requirements are:

- Python 3.8+
- NumPy, Matplotlib (orbital simulation)
- HuggingFace Transformers, NLTK (VeritasVigil NLP pipeline)

## Context

These tasks were submitted as part of the BCS Secretary Recruitment process at IIT Kanpur.
